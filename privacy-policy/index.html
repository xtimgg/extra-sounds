Extra Sounds Extension Privacy Policy

This extension does not:

  • Collect any personal information
  • Track user behavior
  • Store or transmit data to external servers

User preferences, including sound settings and URLs, are stored securely in the browser's local storage using Chrome's sync feature. Audio files are loaded directly from the URLs provided by the user and are not stored locally, unless the user chooses (in options) to cache them. Users can also clear the cache through the options pop-up

Local Files: Users may optionally provide file paths (e.g., file:///) to play local audio. These files are read once for playback and never stored or transmitted.
